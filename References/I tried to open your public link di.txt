## Option B — Save a clean TXT file (downloads automatically)

1. Open the page, press **F12** (or Right-click → Inspect) → **Console**.
2. Paste and run:

```js
(() => {
  const pick = (s) => document.querySelector(s);
  const el = pick('#root') || pick('main') || pick('#app') || document.body;
  // Strip likely nav/footers if present
  el.querySelectorAll('nav,footer,script,style,[aria-hidden="true"]').forEach(n => n.remove());
  const text = el.innerText.replace(/\n{3,}/g, '\n\n').trim();
  const blob = new Blob([text], {type: 'text/plain;charset=utf-8'});
  const a = document.createElement('a');
  a.href = URL.createObjectURL(blob);
  a.download = 'LouisAmy_Proposal_Extract.txt';
  document.body.appendChild(a); a.click(); a.remove();
  URL.revokeObjectURL(a.href);
  console.log('✅ Saved', text.length, 'chars to LouisAmy_Proposal_Extract.txt');
})();
```
